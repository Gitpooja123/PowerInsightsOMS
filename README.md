# PowerInsights OMS

A modern web-based **Outage Management System (OMS)** designed to streamline the process of reporting, tracking, and resolving power-related issues efficiently.

---

## Overview

** PowerInsights OMS** helps users report electricity issues, enables admins to assign technicians, and ensures quick resolution with real-time tracking and notifications.

It bridges the gap between **users, technicians, and administrators** for faster communication and better service management.

## Key Features

### User
- Report power outage issues
- Upload images of the issue 
- Track complaint status
- Provide feedback

### Admin
- Dashboard with analytics
- Assign technicians to complaints
- Monitor outage reports
- Manage users & technicians

### Technician
- Receive assigned tasks
- View issue details
- Update status (Pending → Resolved)

---

## Core Functionalities

- Real-time complaint tracking
- Notification system
- Outage mapping (Live Map)
- Activity logging (Audit trail)
- Dark / Light theme support
- Multi-language support (EN / HI)
